Miles is open source under the LICENSE file in the repo. We accept community contributions of every size.

## Repository layout

The repo is organized as follows: `miles/` contains the main package, `examples/` the recipes, `scripts/` the launchers, and `tests/` the test suite.

## Style

- Python: use `ruff` for linting and `black` for formatting.
- Type hints: use them.
- Docstrings: Google style for public functions.

## Commit messages

Use conventional commits style. Keep the first line under 70 characters.

## Where to ask questions

- Quick questions: Miles channel of the SGLang Slack.
- Security: email security@radixark.ai (do not open a public issue).
